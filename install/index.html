The simplest grammar checker, free to start. Works on Microsoft Edge (and Chrome-compatible browsers).

Install from Edge Add-ons →

Installation — 3 Steps

1Click "Install from Edge Add-ons" above

This opens the Microsoft Edge Add-ons store page for WriteLite. Click the blue "Get" button.

2Click "Add extension"

Edge will ask you to confirm. Click "Add extension." WriteLite only needs permissions for grammar checking — no browsing data access.

3Start using WriteLite

Open any website with a text input (Gmail, Outlook, Slack). Type a sentence. Press the spacebar 3 times quickly. That's it!

How to Use

✍️Type your text → anywhere on any website
␣␣␣Triple-space → press spacebar 3 times quickly
✓Fix All → one click corrects every error

First Time Setup

To use AI grammar checking, you'll need an API key from DeepSeek:

  1. Go to platform.deepseek.com and sign up
  2. Add ¥10 credits (enough for ~1 year of personal use)
  3. Create an API key
  4. In WriteLite: click the icon → Settings → Advanced → paste your API key

Once set up, WriteLite is ready to check your grammar anywhere.
