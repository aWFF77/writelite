Installation — Works in Edge & Chrome

1Click "Install from Edge Add-ons" above

Opens the Microsoft Edge Add-ons store. Works in Edge AND Chrome — both browsers support the same extension format.

2Click "Get" then "Add extension"

In Edge: click "Get" → "Add extension". In Chrome: click "Get" → Chrome will prompt "Add extension from Edge Store?" → click "Add".

3Start using WriteLite

Open any website with a text input (Gmail, Outlook, Slack). Type a sentence. Press the spacebar 3 times quickly. That's it!
